# Security Policy

## Supported Versions

Only the latest release of TypeGuard is actively supported with security updates.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub Issues.

Instead, report them privately via [GitHub's private vulnerability reporting](https://github.com/DoubledDoge/TypeGuard/security/advisories/new).

You can expect an initial response within a few days. If the vulnerability is confirmed, a fix will be prioritised and a patched release will be issued as soon as possible.
